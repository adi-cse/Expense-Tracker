# 💰 Smart Expense Tracker (Java)

A menu-driven Java application to manage personal income and expenses with category-wise tracking and analysis. Built using core Java concepts like OOP, Collections, and File Handling.

---

## 🚀 Features
- ➕ Add Income  
- ➖ Add Expense with Category  
- 📊 View All Transactions  
- 💼 Check Current Balance  
- 💾 Persistent Data Storage using File Handling  
- ⚡ Simple Menu-Driven Interface  

---

## 🛠️ Tech Stack
- Java (Core Java)  
- OOP (Object-Oriented Programming)  
- Java Collections (ArrayList)  
- File Handling (Object Streams)  

---

## 📂 Project Structure
ExpenseTracker/  
│── Main.java  
│── Transaction.java  
│── ExpenseService.java  
│── FileHandler.java  

---

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/your-username/ExpenseTracker.git  

2. Navigate to the project folder:
cd ExpenseTracker  

3. Compile the Java files:
javac *.java  

4. Run the application:
java Main
